Main Changes:

Rav Assistant Version 1.4

1. You can directly click the History link to access the history page.

2. Each RAV run results (column) have 4 links:

All: Show all results, default
Run: Show all run results, except the Not Run results
Fail: Show all fail results(include FAIL,FATAL,INVALID,...), except the Not Run and PASS results
Crash: Show all crash results, except the others

3. You can click the INFO link to access the info page.

4. Right click Fail or Crash case, select History and Result Html to open history and result file directly.

5. Right click Pass case, select Open to open result file directly.
All file links in any pages can use this way to open.

6. Change the bubble popup code from synchronous pattern to asynchronous pattern to improve the block issue of UI.
